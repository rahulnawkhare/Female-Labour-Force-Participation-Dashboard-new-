# 📊 Female Labour Force Participation Dashboard (India)

A Power BI dashboard that analyzes Female Labour Force Participation Rate (LFPR) in India from **2017–18 to 2023–24**.

## Project Overview

This interactive dashboard provides insights into Female Labour Force Participation using multiple dimensions such as:

- Fiscal Year
- Region
- Gender
- Social Group
- Growth Percentage
- Latest LFPR
- Average LFPR

---

## Dashboard Features

- 📈 Growth Percentage KPI
- 👩 Latest Female LFPR
- 📊 Average LFPR
- 📉 LFPR Trend Analysis
- 🌍 Region-wise Analysis
- 👥 Social Group Analysis
- 🎛 Interactive Slicers
- 📅 Fiscal Year Filter

---

## Technologies Used

- Microsoft Power BI
- Microsoft Excel
- DAX
- Data Visualization

---

## Dataset

The dataset contains Female Labour Force Participation information from 2017–18 to 2023–24.

Fields include:

- Fiscal Year
- LFPR (%)
- Region
- Gender
- Social Group

---

## Insights

- Female LFPR shows a consistent increasing trend.
- Rural participation is higher than Urban.
- Social group analysis reveals variations in labour force participation.
- Interactive filters allow users to explore multiple perspectives.
